# Sound Assets

Phase 4 supports these local-only sound keys:

- `chime.mp3`
- `pop.mp3`
- `success.mp3`
- `animals/cow.mp3`
- `animals/dog.mp3`
- `animals/cat.mp3`
- `animals/pig.mp3`
- `animals/duck.mp3`
- `animals/frog.mp3`
- `animals/sheep.mp3`
- `animals/horse.mp3`
- `words/ball.mp3`
- `words/cat.mp3`
- `words/milk.mp3`
- `words/mama.mp3`
- `words/dada.mp3`

The app treats these as optional during development. Missing files must not crash touch interactions.
